# 🌐 GoChat Web Preview

Demostración interactiva de la aplicación GoChat. Esta es una **preview web profesional** que muestra cómo se vería la app mobile en un navegador.

## 📱 Características

- **Simulación de teléfono realista** (390x844px)
- **Pantallas interactivas**: Home, Matches, Premium
- **Animaciones profesionales** con cubic-bezier
- **Efectos visuales** (gradientes, sombras, blur)
- **Navegación fluida** entre pantallas
- **Responsive design** para todos los dispositivos

## 🚀 Cómo Usar

1. Abre `index.html` en tu navegador
2. Interactúa con los botones y elementos
3. Navega entre pantallas con los botones inferiores
4. Prueba en diferentes tamaños de pantalla

## ⌨️ Controles

- **Clic en botones**: Interactúa con elementos
- **Flechas izquierda/derecha**: Navega entre pantallas
- **Hover**: Ve efectos en botones y tarjetas

## 📊 Pantallas Incluidas

| Pantalla | Descripción |
|----------|-------------|
| **Home** | Swipe de perfiles con botones de acción |
| **Matches** | Lista de conversaciones y nuevos matches |
| **Premium** | Planes de suscripción (Gratis, Premium, Gold) |

## 🎨 Diseño

- **Colores**: Magenta (#E91E8C), Violeta (#7C3AED), Negro (#0A0A0F)
- **Tipografía**: Inter (Google Fonts)
- **Tema**: Dark mode premium
- **Animaciones**: Suaves y naturales

## ⚠️ Nota Importante

Esta es una **demostración web estática**. El código real de la app mobile está en `/app` usando **React Native + Expo**.
